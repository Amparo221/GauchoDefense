# GauchoDefense


Proceso de crear el repositorio:
git clone https://github.com/Amparo221/GauchoDefense.git

cd GauchoDefense/

No necesario/solo para checkear el estado:
 git status

git add .

git commit -m 'first commit'

No necesario/solo para checkear el estado:
 git log

git push
